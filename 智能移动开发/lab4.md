## 增加一个按钮

### Jetpack Compose UI 组件
- `Surface`: 提供背景色的 UI 组件。
- `Column`: 用于创建垂直布局的组件。
- `Image`: 用于显示图片资源的组件。
- `Button`: 可点击的按钮组件。
- `Text`: 用于显示文本的组件。

### 状态管理
- `mutableIntStateOf`: 创建一个整数类型的可变状态。

### 资源访问
- `painterResource`: 获取图片资源的画家对象。
- `stringResource`: 获取字符串资源。

### 样式主题
- `DiceRollerTheme`: 自定义应用主题。

### UI 布局与样式
- `Modifier`: 用于修改组件属性的函数集，如大小、填充和对齐。
- `Alignment`: 用于组件对齐的函数。

### 事件处理
- `onClick`: 按钮点击事件处理器。
